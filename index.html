CWL Generator Create CWL files for tools and workflows

This is a simple CWL descriptor generator form. It is meant to be used by beginners of CWL, and as such has been simplified to make it easier to use. It does not support the full set of features offered by CWL, though it should be enough for basic tools.

Please note that this generator does not enforce any of the dependencies that exist within CWL descriptor files. It is up to the user to understand what different settings do and how they will impact other settings.

General Information
